# SimpleX SMP & XFTP Servers - Home Assistant Add-on Repository
A Home Assistant add-on to host [SimpleX Chat](https://simplex.chat) SMP and XFTP servers.

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fed-asriyan%2Fhassio-simplex)

## Add-ons
This repository contains the following add-on:

* [SimpleX SMP Server](./simplex-smp-server)
* [SimpleX XFTP Server](./simplex-xftp-server)
